# 项目简介

## 核心功能
- 宠物领养信息展示与管理
- 用户注册登录与个人信息管理
- 宠物领养申请流程
- 后台管理系统（含超级管理员权限）
- 数据持久化存储与缓存处理

## 适用场景
- 宠物救助机构领养平台
- 社区宠物领养活动
- 校园宠物领养推广
- 宠物爱好者交流社区

## 技术特点
- 采用前后端分离架构，Spring Boot作为后端框架
- 前端使用Vue框架开发，小程序端基于Uniapp跨平台开发
- 使用MyBatis-Plus进行数据持久化操作
- 集成Spring Security实现安全认证
- Redis作为缓存中间件提升系统性能
- MySQL 5.7作为关系型数据库
- 支持在主流IDE（IDEA、VSCode、HBuilder）中开发运行
